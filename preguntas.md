1. ¿Que ventajas observas en la version orientada a objetos?

Encapsulamiento: Los datos del producto están protegidos
Reutilización de código: La clase Producto puede ser util multiples veces
Mantenibilidad: Es más fácil modificar y extender el código
Organización: El código está mejor estructurado y organizado


2. ¿Que parte del codigo resulto mas clara o facil de mantener?

La clase **Producto** con sus métodos calcularTotal() y mostrarInfo() hace que el código sea más claro y se logre entender muy bien, ya que toda la lógica relacionada con un producto está contenida en una sola clase.


3. ¿Como podrıas extender el programa para manejar mas productos?

Se podria implemtenar el uso de un List o ArrayList en lugar de un array fijo
Implementando la clase Tienda que gestione una colección dinámica de productos
Agregando métodos para buscar, eliminar o modificar productos, que haria que se haga más interactivo


4. ¿Podrıas agregar una clase Tienda que contenga los productos y calcule el total?
Sí, como se muestra en el ejemplo anterior. La clase **Tienda** maneja una lista de productos y nos da métodos para gestionar el inventario completo.
